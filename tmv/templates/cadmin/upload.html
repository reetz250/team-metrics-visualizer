File Format Specifications

  • File format: .xlsx or .xls
  • Sheet Names: Year and month in YYMM format meaning each sheet corresponds to a specific month and year.
  • Columns:
    • Code: Required as first column. Team code uniquely identifying the team. Maps to Team model's code field.
    • Overtime: Total time of overtime in hours:minutes format for team represented by Code.
    • Fixed Working days: Number of working days in the month.
    • Actual working days: Number of actual days worked in the month.
